Connection

The port Central Command is listening on. Default is 47293 — only change if you edited the widget settings.

Open Central Command → Job Tracker widget → Extension tab, copy the token, and paste it here. The token authenticates requests so only this extension can add jobs.

Setup guide

  1. Make sure Central Command is running on your desktop.
  2. Open the Job Tracker widget and click the Extension tab.
  3. Copy the authorization token shown there.
  4. Paste it into the Authorization token field above and click Save.
  5. Browse to any job listing, click the extension icon, edit if needed, and hit Add Job.
